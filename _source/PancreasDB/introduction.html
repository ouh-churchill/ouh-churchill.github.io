Introduction¶

This is a Transplant Department project to migrate existing workflow and data into a modern, secure, user-friendly, digital-based process and supporting system. Using an incremental approach to the Pancreas Transplant process, we'll be breaking each aspect down and providing the tools and training to allow staff to upgrade working practices and improve clinical data flows, and ultimately patient experience and outcomes.

Current workflows are being mapped in detail, but are primarily based around paper forms and multiple spreadsheets shared between key users. We need to move this data into a secure and multi-user environment, and start to link up parts of the process, as well as increase automation of data updating and gathering.

Key Objectives¶

  • Removal of all clinical data from local desktops to a maintained service
  • Automation of data gathering to reduce need for rekeying data from other clinical data sources
  • Digitisation of data transfer to NHSBT
  • Provide clinical evaluation overviews for service improvement
